To run the project locally:
1. Install python(version 3.11) and pip(i have version 24.1.2)
2. Run python -m pip install -r requirements.txt
3. Run fastapi dev main.py
4. Go to http://127.0.0.1:8000/docs

Note:
The predict image will saved in results folder, it's ok you can ignore it and just every x time to delete the images.
